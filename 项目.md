# OAuth2是什么？
Oauth2是针对特定问题一种解决方案

令牌机制，按照一定规则生成字符串，字符串包含用户信息


Oauth2主要可以解决两个问题


1. 开放系统间授权

通用开发者key 万能钥匙


办法令牌

管理令牌，办法令牌，吊销令牌需要统一的协议


2. 分布式访问问题

分布式访问单点登录
