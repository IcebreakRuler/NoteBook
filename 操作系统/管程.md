# 管程

## 管程的定义和基本特征
管程是一种特殊的软件模块，有这些部分组成：
1. 局部与管程的共享数据结构说明；
2. 对该数据结构进行操作的一组过程；
3. 对局部于管程的共享数据设置初始值的语句；
4. 管程有一个名字。

### 基本特征
1. 局部于管程的数据只能被局部于管程的过程所访问
2. 一个进程只有通过调用管城内的过程才能进入管程访问共享数据
3. 每次仅允许一个进程在管城内执行某个内部过程
